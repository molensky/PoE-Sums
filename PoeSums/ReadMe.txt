// ReadMe.txt -- Poe Sums
// 
// Rules for Poe Sums:
//
// Purpose is to find the combination to use the least amount of items to make sums of 40.
//
// Items can only be summed once.
// Input will be an array of integers in the range of [1..20].
// Entries outside of the valid input range will be ignored
// Null input will return blank list
// Combinations that don't add up to 40 will be ignored
// An entry of 20 will count as a 40
// Largest Combination Size is 40 (although given input sets should be much smaller)
// Ability to find total of all elements
//
